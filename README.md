# Nginx Log Analyser — Shell Script Project

A lightweight Bash tool to analyse nginx combined access logs.  
It extracts top IPs, top request paths, top response status codes, and top user agents.

## Files
- `nginx-log-analyser.sh` — main script (executable)
- `sample.log` — small demo log for testing

## Usage
Make the script executable (on Linux/macOS):
```bash
chmod +x nginx-log-analyser.sh
